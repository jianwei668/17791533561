# 服务业选址分析系统

一个基于Web的服务业选址分析系统，使用Leaflet地图库开发，具有加载卫星影像、测量和标注地址、导入导出KML文件等功能。

## 功能特点

- 加载卫星影像、街道地图和混合地图
- 默认显示指定区域（纬度: 22.162924-22.186610, 经度: 113.247414-113.304234）
- 标注点、线、面等地理要素
- 测量距离和面积
- 导入导出KML文件
- 扁平化风格UI设计

## 使用方法

1. 直接在浏览器中打开`index.html`文件
2. 使用左侧工具栏进行标注和测量
3. 使用顶部工具栏切换底图、导入导出KML文件
4. 右侧面板显示图层列表、信息和测量结果

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Leaflet 地图库
- Leaflet.draw 绘制插件
- toGeoJSON 和 tokml 用于KML文件转换

## 部署说明

本项目为纯前端应用，不需要服务器环境，只需将所有文件放在同一目录下，然后用浏览器打开`index.html`即可。
